# 100 NumPy Problems

A collection of my solutions to the famous 100 NumPy Exercises challenge. This repository documents my learning journey with NumPy and helps me strengthen the fundamentals required for Data Science, Machine Learning, and Artificial Intelligence.

## About the Challenge

NumPy is the core library for numerical computing in Python. The 100 NumPy Problems challenge covers a wide range of concepts, from basic array operations to advanced numerical computations.

Through this challenge, I am improving my understanding of:

* Array creation and manipulation
* Indexing and slicing
* Vectorized operations
* Broadcasting
* Mathematical functions
* Linear algebra
* Random number generation
* Performance optimization
* Real-world data handling

Technologies Used

* Python 3
* NumPy
* Jupyter Notebook
